3.11. Задание: оптимизация проекта под протокол HTTP/2

Ниже приведена структура файлов проекта, который оптимизирован под HTTP/1. Предложите способы оптимизации под HTTP/2. Наше решение вы найдёте далее.

Структура файлов:

index.html
catalog.html
product.html
img
  logo.svg
  sprite.svg
css
  style.min.css
js
  scripts.min.js

В проекте три страницы: главная, каталог и страница товара. На главной товаров нет, но на ней есть модальное окно.

Весь CSS проекта собирается в один CSS-файл.

Написанный JavaScript-код:

  1. Открытие и закрытие меню для мобильной версии.
  2. Скрипт для добавления товара в корзину.
  3. Скрипт для добавления товара в избранное.
  4. Скрипт для карусели фотографий товара на странице товара.
  5. Скрипт для фильтрации на странице каталога.
  6. Скрипт открытия модального окна.

Весь JavaScript-код собирается в один файл.

Иконки собраны в единый спрайт.

Дальше вас ждёт тест, в котором нужно выбрать способы оптимизации под HTTP/2.
